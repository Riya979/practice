Overflow property:

This property specifies whether to clip the content or to add scrollbars when the content of an element is too big to fit in the specified area.
Among some categories which are further classified the first is represented and shown below:

Auto overflow:

The first property is auto overflow which is similar to scrollbar but only it adds when neccsary.
