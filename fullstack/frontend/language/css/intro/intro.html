CSS-cascading style sheet

CSS is the language we use to style an HTML document.
C-FALLING OF STYLES.
S-ADDING DESIGN
S-WRITING OUR STYLE IN DIFFERENT DOCUMENTS.

Basic Structure:

selector { property1:value; property2:value; } Selector-selects element you want to target like tags,ids,classes.

**key value pair

Property-is nothing but attributes like color,font,background.

Value-associated with the properties.
